## 📦 安装方式

### 方式一：从源码编译

```bash
# 克隆项目
git clone https://github.com/Sunrisies/lan-share.git

# 进入目录
cd lan-share

# 编译运行
cargo run
```

### 方式二：下载预编译版本

访问 [GitHub Releases](https://github.com/...) 下载对应平台的预编译版本。

### 方式三：Docker 部署

```bash
docker run -p 3000:3000 lan-share
```
