# 局域网聊天工具 - 产品白皮书

## 执行摘要

在数字化办公时代，团队协作效率至关重要。局域网聊天工具是一款专为局域网环境设计的即时通讯解决方案，它以简洁、高效、安全为核心理念，为团队提供无缝的沟通体验。

## 部署方案

### 单机部署

适用于小型团队（1-10人）：

```bash
# 在一台电脑上启动
cargo run

# 其他电脑访问
http://192.168.1.100:3000
```

### 服务器部署

适用于中型团队（10-50人）：

```bash
# 编译发布版本
cargo build --release

# 后台运行
nohup ./target/release/lan-share > server.log 2>&1 &
```

## 客户案例

### 案例 1：小型办公室

**场景**：10人设计团队，需要快速分享设计稿

**解决方案**：
- 在办公室电脑上部署
- 团队成员通过浏览器访问
- 实时分享设计稿和反馈

**效果**：
- 文件分享效率提升 80%
- 沟通成本降低 50%

### 案例 2：教育培训

**场景**：计算机教室，需要分发作业和资料

**解决方案**：
- 在教师电脑上部署
- 学生通过局域网访问
- 教师分发文件，学生提交作业

**效果**：
- 文件分发时间从 10 分钟缩短到 1 分钟
- 学生参与度提高
